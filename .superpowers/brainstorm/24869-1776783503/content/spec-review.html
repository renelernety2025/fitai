Tier 1 Features — Design Spec

Zkontroluj a potvrd, nebo rekni co zmenit

1. AI Chat Coach /ai-chat

Layout: Hero + Chat

  • Hero nahore: Alex avatar (gradient kruh), jmeno, popis, 3-4 suggested prompt pills
  • Chat zona: scrollovatelne bubliny (user = gradient vpravo, Alex = tmava vlevo s avatarem)
  • Input dole: rounded input + send button, Enter = odeslat, Shift+Enter = novy radek
  • Streaming pres SSE — text se objevuje token po tokenu
  • Hero se sroluje nahoru jak konverzace roste

Backend (3 nove endpointy):

  • POST /api/coaching/chat — SSE streaming, conversationId pro memory (poslednih 20 zprav jako Claude kontext)
  • GET /api/coaching/conversations — seznam konverzaci uzivatele
  • GET /api/coaching/conversations/:id/messages — zpravy jedne konverzace
  • Auto-title po prvni odpovedi (3-5 slov)
  • Schema: pridat title field na CoachingSession

System prompt:

  • Sdileny buildUserPromptContext() (vek, zraneni, cil, streak, vybaveni)
  • Delsi odpovedi nez pri workoutu (max 200 slov vs 12 slov)
  • Markdown OK, konverzacni ton, prompt injection ochrana

Nove soubory:

  • apps/web/src/app/(app)/ai-chat/page.tsx
  • apps/web/src/components/chat/ChatBubble.tsx
  • apps/web/src/components/chat/ChatInput.tsx
  • apps/api/src/coaching/dto/chat-message.dto.ts
  • apps/api/src/coaching/coaching-chat-prompt.ts

2. Workout Streak Calendar /habity

Komponenta: GitHub-style ActivityHeatmap

  • Mrizka 7 radku (Po-Ne) x 12 sloupcu (12 tydnu = 3 mesice)
  • Bunka ~14x14px, gap 3px, zaoblene rohy
  • Barva podle recovery score: zadny check-in 0-40 41-60 61-80 81-100
  • Dnesek: bily border ring
  • Hover tooltip: datum + recovery score
  • Pod heatmapou: "X check-inu · Nejdelsi streak: Y dni · Aktualni: Z dni"

Data: Existujici getHabitsHistory(84) — jen zvysit parametr z 14 na 84

Novy soubor: apps/web/src/components/habits/ActivityHeatmap.tsx

3. "Co dnes?" Smart Widget /dashboard

Umisteni: Nova karta NAD Daily Brief — prvni vec na dashboardu

Vizual:

  • Gradient background (meni se podle typu akce)
  • Headline (1 veta, max 60 znaku) + rationale (1 veta)
  • CTA button + dismiss (x) — po dismissu schovano do konce dne

Logika doporuceni (priorita):

  1. Streak v ohrozeni (streak > 3, dnes necvicil) → micro-workout
  2. Vysoka bolestivost (soreness ≥ 4) → recovery den
  3. Dlouho necvicil (3+ dny) → lehky trenink
  4. Slaba vyziva (kcal < 70% cile) → doplnit bilkoviny
  5. Vse OK → motivacni akce z Daily Brief

Backend: GET /api/ai-insights/today-action

  • Rules-based, zadny Claude call (instant response)
  • Cache 1h per user

Nove soubory:

  • apps/web/src/components/dashboard/TodayActionCard.tsx
  • apps/api/src/ai-insights/today-action.ts
